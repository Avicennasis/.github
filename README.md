# .github

Default [community health files](https://docs.github.com/en/communities/setting-up-your-project-for-healthy-contributions/creating-a-default-community-health-file) and profile page for [@Avicennasis](https://github.com/Avicennasis).

Files in this repository are automatically inherited by all repositories
under this account that don't provide their own copy:

- **CODE_OF_CONDUCT.md** — Contributor Covenant v2.1
- **CONTRIBUTING.md** — contribution guidelines and PR checklist
- **SECURITY.md** — vulnerability reporting policy
- **FUNDING.yml** — sponsor links
- **Issue & PR templates** — bug reports, feature requests, PR checklist

The `profile/README.md` powers the profile page shown at
[github.com/Avicennasis](https://github.com/Avicennasis).

## Overriding defaults

Any repository can override these files by placing its own copy at the
same path. For example, a repo with a language-specific `CONTRIBUTING.md`
will use its own version instead of the default here.
