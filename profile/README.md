## Hey, I'm Avic

**Infrastructure tinkerer, MCP builder, and volunteer firefighter from Pittsburgh, PA.**

I build tools that solve real problems — from Model Context Protocol servers and sysadmin automation to desktop apps and browser extensions. Most of what's here grew out of my homelab or a fire department that needed better software.
